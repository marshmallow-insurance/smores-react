# < Icon />

## Props:

```ts
render:   string; 
size?:    number; /* box dimentions */
color?:   string; /* from theme */
rotate?:  number; /* degrees */
```

## Usage

```js
import {Icon} from '@mrshmllw/smores-react';

const App = () => (
  <Icon render='edit' color="pink2" size={24} />
);
```
