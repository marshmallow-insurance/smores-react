# < Icon />

## Props:

```ts
render:   string; 
size?:    number; /* icon dimentions */
color?:   string; /* from theme */
rotate?:  number; /* degrees */
```

## Usage

```js
import {Icon} from '@mrshmllw/smores-react';

const App = () => (
  <Icon render='edit' color="marshmallowPink" size={24} />
);
```
