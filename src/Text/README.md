# < Text />

## Props:

```ts
tag:        any;    /* HTML tag    */
className?: string;
typo?:      Typo | (string & Record<never, never>); /* typography  */
align?:     string; /* text-align  */
color?:     Color; /* from theme */
cursor?:    string;
```

## Usage

```js
import {Text} from '@mrshmllw/smores-react';

const App = () => (
  <Text tag='h2' typo='hero-alternate' color='secondary'>Hey you</Text>
);
```
